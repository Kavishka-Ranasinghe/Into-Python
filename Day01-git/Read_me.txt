#### Day 01 - Random number generator ####
Welcome to Random number generator game.
This is a simple python program consist of following topics
            # Variable
            # In-built functions
            # Loops
            # User defined  functions
            # IF,ELIF,ELSE Condition

This game simple it has 2 players. User needs to enter letter "g" to continue and press "e" to exit.
Each player gets 5 chance (10 total) each time random number from 1-10 will be generated and saved in the variable.
Finally, the player with the highest score wins the game.
If you exit the game while in process current marks will not be shown, you just exit the program :(
This just Day 01, so it is not advance just to cover basics of python :).
